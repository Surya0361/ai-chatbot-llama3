# AI Chatbot using Llama3

A local AI chatbot built using **Streamlit**, **Ollama**, and **Llama3**.

This project runs a Large Language Model locally without using external APIs.

## Features
- ChatGPT-like chatbot interface
- Runs completely offline using Ollama
- Built with Python and Streamlit
- Simple and lightweight AI assistant

## Technologies Used
- Python
- Streamlit
- Ollama
- Llama3

## Installation

### 1. Install Ollama
Download and install Ollama from:
https://ollama.com/

### 2. Pull the Llama3 model
ollama pull llama3
### 3. Install Python dependencies
pip install -r requirements.txt
### 4. Run the chatbot
streamlit run app.py
